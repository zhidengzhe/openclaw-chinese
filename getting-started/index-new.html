快速开始

安装 OpenClaw，运行初始化设置，并与您的 AI 助手聊天 — 整个过程大约只需 5 分钟。

您需要什么

  • Node.js — 推荐 Node 24（也支持 Node 22.14+）
  • API 密钥 — 来自模型提供商（Anthropic、OpenAI、Google 等）

快速安装

macOS / Linux

curl -fsSL https://openclaw.ai/install.sh | bash

Windows (PowerShell)

iwr -useb https://openclaw.ai/install.ps1 | iex

下一步

  1. 运行初始化：openclaw onboard
  2. 启动网关：openclaw gateway start
  3. 开始聊天：openclaw chat

这是一个简化版本，用于测试编码是否正确。
