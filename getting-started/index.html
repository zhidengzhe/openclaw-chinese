快速开始

安装 OpenClaw，运行初始化设置，并与您的 AI 助手聊天 — 整个过程大约只需 5 分钟。结束时，您将拥有一个运行的网关、配置好的身份验证和一个正常工作的聊天会话。

您需要什么

  • Node.js — 推荐 Node 24（也支持 Node 22.14+）
  • API 密钥 — 来自模型提供商（Anthropic、OpenAI、Google 等）— 初始化设置时会提示您

提示

使用 node --version 检查您的 Node 版本。

Windows 用户：支持原生 Windows 和 WSL2。WSL2 更稳定，推荐用于完整体验。请参阅 Windows 平台。

需要安装 Node？请参阅 Node 安装指南。

快速安装

步骤 1：安装 OpenClaw

macOS / Linux

curl -fsSL https://openclaw.ai/install.sh | bash

Windows (PowerShell)

iwr -useb https://openclaw.ai/install.ps1 | iex

注意

其他安装方法（npm、Docker 等）请参阅完整的 安装指南。

步骤 2：运行初始化设置
# 运行初始化向导 openclaw onboard

初始化向导将：

  • 引导您完成基本配置
  • 提示您输入 API 密钥
  • 设置默认模型
  • 配置网关服务
步骤 3：启动网关
# 启动网关服务 openclaw gateway start

或者手动运行：

# 手动运行网关（开发模式） openclaw gateway
步骤 4：开始聊天
# 启动聊天会话 openclaw chat

您现在可以与您的 OpenClaw 助手聊天了！

下一步

  • 连接消息渠道 — 设置 WhatsApp、Telegram 或 Discord 集成
  • 探索技能 — 安装天气、日历、电子邮件等技能
  • 自定义配置 — 调整设置以满足您的需求
  • 加入社区 — 获取帮助并分享您的体验

故障排除

如果遇到问题：

  1. 检查网关是否运行：openclaw gateway status
  2. 查看日志：openclaw logs
  3. 验证 API 密钥配置
  4. 访问 帮助页面 获取常见问题解答

需要帮助？

如果您在快速开始过程中遇到困难：

  • 查看详细的 安装指南
  • 访问 帮助中心
  • 加入 Discord 社区
  • 在 GitHub Issues 中报告问题
