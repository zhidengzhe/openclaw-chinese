帮助

常见问题解答和故障排除指南。

常见问题

安装问题

Q: 安装时出现权限错误？

A: 尝试使用管理员权限运行：sudo npm install -g openclaw@latest

Q: Node.js 版本要求？

A: 需要 Node.js 22.14+ 或 24.x

运行问题

Q: 网关无法启动？

A: 检查端口 18789 是否被占用：netstat -ano | findstr :18789

Q: 控制面板无法访问？

A: 确保网关正在运行：openclaw status

配置问题

Q: 如何配置 API 密钥？

A: 在配置文件中设置或使用环境变量：OPENCLAW_API_KEY=your-key

故障排除

  1. 检查日志：openclaw logs
  2. 重启服务：openclaw restart
  3. 更新到最新版本：npm update -g openclaw
  4. 查看官方文档或社区寻求帮助
