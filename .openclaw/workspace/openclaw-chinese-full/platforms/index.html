平台

OpenClaw 支持多种平台和操作系统。

支持平台

  • macOS - 完整支持，包括原生应用
  • Windows - PowerShell 和 WSL2 支持
  • Linux - 所有主要发行版
  • iOS - 移动节点应用
  • Android - 移动节点应用
  • Docker - 容器化部署
  • 云平台 - AWS, Azure, GCP, DigitalOcean 等

平台特定功能

  • macOS - 菜单栏应用、Canvas、语音唤醒
  • Windows - PowerShell 集成、WSL2 优化
  • 移动端 - 摄像头访问、位置服务、语音输入
  • 服务器 - 24/7 运行、远程访问、监控
