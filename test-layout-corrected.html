快速开始

安装 OpenClaw，运行 onboarding，并与您的 AI 助手聊天 — 全部大约需要 5 分钟。最后，您将拥有一个运行的网关、配置的身份验证和一个正常工作的聊天会话。

您需要什么

Node.js — 推荐 Node 24（也支持 Node 22.14+）

模型提供商的 API 密钥（Anthropic、OpenAI、Google 等）— onboarding 将提示您

快速设置

安装 OpenClaw

macOS / Linux: curl -fsSL https://openclaw.ai/install.sh | bash

Windows (PowerShell): iwr -useb https://openclaw.ai/install.ps1 | iex

运行 onboarding

openclaw onboard --install-daemon

向导将引导您选择模型提供商、设置 API 密钥和配置网关。大约需要 2 分钟。

验证网关运行状态

openclaw gateway status

您应该看到网关在端口 18789 上监听。

打开控制面板

openclaw dashboard

这将在浏览器中打开控制 UI。如果加载成功，说明一切正常。

发送第一条消息

在控制 UI 聊天中输入消息，您应该会收到 AI 回复。
