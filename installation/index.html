安装指南

系统要求

在安装 OpenClaw 之前，请确保您的系统满足以下要求：

基本要求

  • Node.js: 版本 18 或更高
  • npm: 版本 8 或更高
  • 操作系统: Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+, Fedora 34+, 等)
  • 内存: 至少 4GB RAM
  • 存储空间: 至少 2GB 可用空间

可选依赖

  • Docker: 用于容器化部署
  • Git: 用于从源码安装
  • Python 3: 某些技能可能需要

Windows 安装

1. 安装 Node.js

访问 Node.js 官网 下载 Windows 安装包，选择 LTS 版本。

2. 安装 OpenClaw

打开 PowerShell 或命令提示符，运行：

npm install -g openclaw@latest

3. 验证安装

openclaw --version

应该显示 OpenClaw 的版本号。

注意：如果遇到权限问题，可以尝试以管理员身份运行 PowerShell。

macOS 安装

1. 安装 Homebrew (可选)

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. 安装 Node.js

使用 Homebrew 安装：

brew install node

或从官网下载安装包。

3. 安装 OpenClaw

npm install -g openclaw@latest

Linux 安装

Ubuntu/Debian

# 安装 Node.js curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - sudo apt-get install -y nodejs # 安装 OpenClaw sudo npm install -g openclaw@latest

Fedora/RHEL

# 安装 Node.js curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash - sudo dnf install -y nodejs # 安装 OpenClaw sudo npm install -g openclaw@latest

Docker 安装

1. 安装 Docker

参考 Docker 官方文档 安装 Docker。

2. 运行 OpenClaw 容器

docker run -d \ --name openclaw \ -p 18789:18789 \ -v openclaw-data:/data \ ghcr.io/openclaw/openclaw:latest

3. 访问控制界面

打开浏览器访问：

http://localhost:18789

验证安装

检查版本

openclaw --version

查看帮助

openclaw --help

运行引导设置

openclaw onboard --install-daemon

如果以上命令都能正常执行，说明安装成功。

故障排除

权限被拒绝

问题：安装时出现 "EACCES: permission denied" 错误

解决方案：

# 使用 sudo (Linux/macOS) sudo npm install -g openclaw@latest # 或修复 npm 权限 npm config set prefix ~/.npm-global echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc source ~/.bashrc

命令未找到

问题：安装后运行 openclaw 命令提示未找到

解决方案：确保 npm 的全局 bin 目录在 PATH 环境变量中

端口被占用

问题：启动网关时端口 18789 被占用

解决方案：使用其他端口

openclaw gateway --port 18790
