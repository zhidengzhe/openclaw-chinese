网关服务安装（CLI）

使用以下方法之一（全部支持）：

  • 向导（推荐）：openclaw onboard --install-daemon
  • 直接安装：openclaw gateway install
  • 配置流程：openclaw configure → 选择 网关服务
  • 修复/迁移：openclaw doctor（提供安装或修复服务的选项）

服务目标取决于操作系统：

  • macOS：LaunchAgent
  • Linux：systemd
  • Windows：Windows 服务

提示

平台选择建议：

  1. 开发环境：使用 macOS 或 Linux
  2. 生产环境：使用 VPS 或云托管
  3. 移动使用：使用 iOS/Android 配套应用
  4. Windows 用户：使用 WSL2 运行网关
