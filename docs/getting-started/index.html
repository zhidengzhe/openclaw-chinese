快速开始指南

OpenClaw 快速开始指南，帮助你在几分钟内启动和运行。

快速开始指南 🚀

本指南将帮助你在几分钟内启动和运行 OpenClaw。

系统要求

支持的操作系统

  • Windows 10/11（推荐）
  • macOS 10.15+
  • Linux（Ubuntu/Debian/CentOS等）
  • 树莓派 Raspberry Pi OS

硬件要求

  • 内存：至少 2GB RAM
  • 存储：至少 500MB 可用空间
  • 网络：稳定的互联网连接

软件依赖

  • Node.js 18.x 或更高版本
  • npm 或 pnpm（推荐 pnpm）

安装 OpenClaw

Windows 用户

# 1. 安装 Node.js（如果还没有）
# 访问 https://nodejs.org/ 下载安装

# 2. 安装 OpenClaw
npm install -g openclaw@latest

# 3. 验证安装
openclaw --version

macOS 用户

# 使用 Homebrew 安装
brew install openclaw/tap/openclaw

# 或者使用 npm
npm install -g openclaw@latest

Linux 用户

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
npm install -g openclaw@latest

# CentOS/RHEL
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo yum install -y nodejs
npm install -g openclaw@latest

首次设置

运行引导向导

openclaw onboard --install-daemon

这个交互式向导会帮你：

  1. 配置网关设置
  2. 创建工作区
  3. 安装必要技能
  4. 设置系统服务（可选）

基本配置检查

# 检查系统状态
openclaw status

# 运行诊断
openclaw doctor

# 查看网关状态
openclaw gateway status

启动网关

作为前台进程运行

# 启动网关（前台运行）
openclaw gateway --port 18789

作为系统服务运行

# 安装为系统服务
openclaw gateway install

# 启动服务
openclaw gateway start

# 检查服务状态
openclaw gateway status

访问控制界面

网关启动后，打开浏览器访问控制界面：

  • 本地默认地址：http://127.0.0.1:18789/
  • 远程访问：需要额外配置

控制界面提供：

  • 聊天界面
  • 会话管理
  • 配置设置
  • 系统监控

基本使用

发送第一条消息

# 通过命令行发送消息
openclaw message send --target +15555550123 --message "你好，OpenClaw！"

检查会话

# 查看当前会话
openclaw sessions

# 查看日志
openclaw logs --limit 20

管理模型

# 检查模型状态
openclaw models status

# 列出可用模型
openclaw models list

配置渠道

配置 WhatsApp

# 登录 WhatsApp
openclaw channels login --channel whatsapp

# 检查渠道状态
openclaw channels status --probe

配置 Telegram

# 添加 Telegram 机器人
openclaw channels add --channel telegram --token "你的机器人令牌"

常见问题

安装问题

# 如果安装失败，尝试清理缓存
npm cache clean --force

# 重新安装
npm install -g openclaw@latest

启动问题

# 检查端口是否被占用
netstat -ano | findstr :18789  # Windows
lsof -i :18789                 # macOS/Linux

# 运行诊断
openclaw doctor --fix

网络问题

# 检查网关连接
openclaw gateway status --deep

# 查看详细日志
openclaw logs --limit 50

提示和技巧

性能优化

  1. 使用 pnpm 代替 npm（更快）
  2. 确保足够的磁盘空间
  3. 保持稳定的网络连接
  4. 定期清理日志文件

安全建议

  1. 不要使用 root 权限运行日常命令
  2. 定期备份配置文件
  3. 保护 API 密钥，不要分享
  4. 定期更新到最新版本

中国用户特别提示

  1. 使用淘宝镜像加速下载
  2. 可能需要配置网络代理
  3. 某些 API 服务可能需要特殊配置
